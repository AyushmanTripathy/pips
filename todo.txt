TOKEN TYPES
-----------
0 -> name types
-1 -> number types
-2 -> string types
-3 -> boolean type
-5 -> null type

-10 -> function types
-11 -> def functions
-12 -> def patterns
-13 -> self call
-15 -> return value

-21 -> if
-22 -> elif
-23 -> else

+ve number -> ascii number of special characters
-100 + keywordIndex -> keywords
